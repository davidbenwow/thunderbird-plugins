Install OmniReply

A quick setup guide for editors. Download the extension, install it in Thunderbird, then connect it with your V4 API key and Cover Playground link.

Latest version: 4.0.164. Versioned file: omnireply-4.0.164.xpi.

Before you begin

  • 1Thunderbird 140 or newer
  • 2Your V4 API key from the V4 account page
  • 3Your personal Cover Playground link for each imprint account
  • 4About 5 minutes for first-time setup

Setup overview

  • ✓Download the XPI file
  • ✓Copy your V4 API key
  • ✓Allow this internal add-on in Thunderbird
  • ✓Install the XPI file
  • ✓Paste your API key and Cover Playground links
1

Download the extension file

Click Download OmniReply. Keep the downloaded XPI file available for the install step.

2

Copy your V4 API key

Open v4.vdm-vsg.de, go to Account, and copy the API key below your signatures.

3

Open Thunderbird settings

  • In Thunderbird, open Thunderbird → Settings.
  • Choose General.
  • Scroll all the way down and click Config Editor.
4

Allow the internal XPI

  • Search for xpinstall.signatures.required.
  • Make sure it is set to false.
  • If it is true, double-click the row to change it to false.

Only change this setting for the internal OmniReply XPI installation. Do not install add-ons from unknown sources.

5

Install the XPI file

  • Open Tools → Add-ons and Themes.
  • Click the settings gear.
  • Choose Install Add-on From File....
  • Select the XPI file you downloaded.
6

Activate OmniReply

Open OmniReply settings, paste your V4 API key, and click Test connection. The status should show Ready.

7

Add your Cover Playground links

Paste your personal Cover Playground link for each recognized imprint account. OmniReply uses these links when a reply should include your submission link.

After setup

Open an email in Thunderbird and click OmniReply. Read the generated answer before using it, especially when the reply contains links, names, prices, or imprint-specific information.
